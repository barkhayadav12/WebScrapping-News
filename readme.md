# Installation and Setup
```js
Ensure that you have python and pip installed in your device

Steps to run it:
1.Run "pip install flask requests beautifulsoup4 flask-cors" in your terminal 
2. Run "python app.py" in your terminal (You will see running on http://127.0.0.1:5000/ in your terminal)
3. Open index.html in your browser or navigate to http://127.0.0.1.5000/

If you want to see the json format, navigate to http://127.0.0.1:5000/news
```
